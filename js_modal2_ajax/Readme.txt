Script permettant au modal l'utilisation du clavier pour la fermeture 
ou la navigation à l'interieur de celui-ci.
Et possibilité d'ouvrir un modal existant dans une page differente(ajax)

vous trouverez les fichier suivant :
    * html 
    * css3
    * Javascript

pour l'utiliser, rien de plus simple copier le html inséré-le dans votre projet,idem pour le css 
et copier le fichier app.js et collé-le dans votre dossier Javascript. 

merci à vous d'utiliser mes scripts. 👍
